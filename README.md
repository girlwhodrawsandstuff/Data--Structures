# Data-Structures
Data Structures implemented in C.

## Topics covered: ##
1. Arrays
  - sparse matrix using arrays
  - lists using arrays
2. Lists
  - singly linked lists
  - doubly linked lists
  - circular linked lists
3. Stacks
  - stack using arrays
  - stack using linked lists
4. Queues
  - linear queue using array
  - circular queue using array
  - queue using singly linked list
